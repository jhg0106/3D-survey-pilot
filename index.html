1 드론 시뮬레이터 앱 제작 흐름 (Stack & Architecture)

본 체험형 시뮬레이터는 웹 표준 기술과 드론 매핑 알고리즘이 융합된 고성능 애플리케이션입니다. 다음 핵심 구성 요소로 설계되었습니다:

3D 렌더링 엔진

Three.js를 사용하여 고성능 웹 지형, 드론 메쉬, 카메라 절두체(Frustum), 웨이포인트(GCP)를 정밀하게 실시간 시각화합니다.

측량 경로 계산 알고리즘

비행 고도 및 중첩도(Front/Side Overlap)를 기반으로 GSD(Ground Sampling Distance)를 계산하고, Lawnmower(지구전체 격자) 자율비행 경로를 동적 자동 연산합니다.

인터랙티브 UI

Tailwind CSS와 Web Audio API를 활용하여 셔터 클릭 사운 피드백 및 모던 글래스모피즘 HUD 측량 패널을 구현했습니다.

3 초간단 무료 웹 배포 (Deployment) 프로세스

구현 완료한 시뮬레이터를 전 세계 어디서나 모바일/PC로 접속할 수 있도록 서버 비용 없이 배포하는 가장 대표적인 방법 3가지를 설명합니다:

방법 1. GitHub Pages (완전 무료, 정적 웹에 초간단 추천)

  1. GitHub 계정을 생성하고 새 리포지토리(Repository)를 만듭니다 (예: drone-survey-sim).
  2. 위 시뮬레이터 코드를 복사하여 index.html 파일로 저장한 후 리포지토리에 푸시(Push) 또는 업로드합니다.
  3. 리포지토리 설정 [Settings] → [Pages] 메뉴로 들어갑니다.
  4. Build and deployment 소스를 Deploy from a branch로 지정하고, Branch를 main (혹은 master) 및 / (root)폴더로 선택 후 Save를 클릭합니다.
  5. 1~2분 대기하면 https://{본인ID}.github.io/drone-survey-sim/ 주소로 즉시 전세계 라이브 배포 완료!

방법 2. Vercel 배포 (가장 빠르고 트렌디하며 React 프로젝트에 최적)

  1. Vercel 사이트에 가입한 후 [Add New Project]를 선택합니다.
  2. 본인의 GitHub 계정을 임포트(Import)하여 리포지토리와 연동합니다.
  3. 프로젝트 빌드 설정은 기본값으로 두고 [Deploy] 버튼을 원클릭합니다.
  4. 자동으로 가상 CDN 서버에 빌드 업로드되며 약 30초 안에 커스텀 도메인(Vercel 무료 하위도메인)이 주어집니다.
